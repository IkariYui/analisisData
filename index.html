Descripción AppWeb

Una AppWeb destinada al manejo de datos de la base de datos de CorAntioquia cuyo enfoque es el desarrollo y protección del medio ambiente. En la App se podrán encontrar recursos para el análsis de datos sobre esta herramienta, y también hacer uso de ellos, importar tus datos para analizaros, y si desea también exportarlos.
En la opción de importar datos se podrá hacer uso de éste medio para que el usuario tenga la posibilidad de utilizar sus propios datos para analizarlos y usar dichos resultados de la manera que crea oportuna.
Al momento usar la herramienta de analizar los datos se podrán ver informes y gráficas al detalle de los datos que se utilizaron, como ejemplos básicos se podrían ver la media, la mediana y la moda en gráficos de barras representando sus datos.
Haciendo uso de la herramienta para exportar los datos podrá tener la posibilidad de guardar los resultados del análisis de sus datos y usarlos de manera oportuna en la forma que considere necesario.
